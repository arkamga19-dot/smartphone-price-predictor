# 📱 Prédicteur de Prix de Smartphones

## Description
Projet de Machine Learning combinant du web scraping et un modèle de prédiction pour estimer le prix des smartphones en fonction de leurs caractéristiques techniques.

## 🛠️ Technologies utilisées
- **Python** — langage principal
- **BeautifulSoup / Requests** — web scraping
- **Pandas / NumPy** — traitement des données
- **Scikit-learn** — modèle Random Forest
- **Matplotlib** — visualisation des résultats

## 📂 Structure du projet
- `scraper.py` — collecte des données sur le web
- `data_collector.py` — nettoyage et préparation des données
- `modèle.py` — entraînement du modèle Random Forest
- `smartphones.csv` — dataset des smartphones
- `résultats.png` — visualisation des performances du modèle

## 🚀 Résultats
Modèle Random Forest entraîné sur des données réelles de smartphones pour prédire leur prix en fonction des specs techniques (RAM, stockage, marque, etc.).
